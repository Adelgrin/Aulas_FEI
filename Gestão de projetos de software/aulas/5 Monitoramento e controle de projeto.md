## Indicadores essenciais para tratamento de desvio 
### Tradicional
#### **AVA** (Análise de valor agregado)
	Técnica de monitoramento de projetos que compara Planejamento X Entrega
##### Variáveis
* **VP**  = planejamento (Valor monetário do planejamento)
* **VA**  = entrega (valor do custo monetário multiplicado pela porcentagem {exemplo R$ 3000 \* 0.7})
* **CA**  = gasto (Custo do planejamento + o custo de dias extras trabalhados)
##### Índice de desempenho de custo
	Mede eficiência de custo
	
$$
IDC = \dfrac{VA}{CA}
$$
* **IDC = 1** : O projeto está dentro do orçamento
* **IDC > 1** : O projeto está a baixo do orçamento * **IDC < 1** : O projeto está a cima do orçamento
##### Índice de desempenho do cronograma 
	Mede eficiência do cronograma

$$
IDCRO = \dfrac{VA}{VP}
$$
* **IDCRO = 1** : O projeto está dentro do cronograma
* **IDCRO > 1** : O projeto está adiantado
* **IDCRO < 1** : O projeto está atrasado
##### Tempo estimado para conclusão do projeto
$$
Tempo\ estimado\ para\ concluir\ o\ projeto = \dfrac{Tempo\ planejado\ para\ o\ projeto}{IDCRO}
$$
##### Prevendo o futuro
$$
Estimativa\ no\ termino = \dfrac{Orçamento\ Total planejado}{IDC}
$$
