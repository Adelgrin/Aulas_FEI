# 2 Reuso de software

## Onde utilizar:

Reultilização de software pode ser ultilizado em:

* Componentes
* Interfaces
* Entre componentes (Ultilização de APIs)

## Outros objetos a se reultilizar:

* Reuso de aplicações (software completo)
* Reuso de componentes
* Reuso de objetos ou funções
